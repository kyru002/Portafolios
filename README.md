# ✨ Mi Portafolio Profesional — Enrique Abad

¡Bienvenido al repositorio de mi portafolio profesional! Esta es una aplicación web moderna, interactiva y de alto rendimiento diseñada para presentar mis habilidades, estadísticas de desarrollo, experiencia y proyectos como **Desarrollador Web Full Stack y Especialista en Inteligencia Artificial**.

🌐 **Ver el portafolio en vivo:** [https://kyru002.github.io/Portafolios/](https://kyru002.github.io/Portafolios/)

---

## 🚀 Tecnologías Principales

*   **Framework:** [Vue 3](https://vuejs.org/) (usando la API `<script setup>` reactiva y moderna).
*   **Herramienta de Construcción:** [Vite](https://vite.dev/) (para un entorno de desarrollo ultra rápido y builds optimizados).
*   **Estilos:** CSS nativo personalizado, con efectos modernos de glassmorphism, degradados vibrantes y micro-animaciones fluidas.
*   **Despliegue:** Integración continua con [GitHub Actions](https://github.com/features/actions) para despliegues automáticos directos a **GitHub Pages**.

---

## ✨ Características Destacadas

*   **Diseño Interactivo y Fluido:** Incluye fondos animados dinámicos, efectos interactivos de inclinación 3D (*tilt*) al mover el ratón sobre las tarjetas de proyectos y transiciones de carrusel cuidadas.
*   **Galería de Proyectos Dinámica:**
    *   Carrusel interactivo para navegar por los proyectos realizados.
    *   Previsualización a pantalla completa de imágenes mediante un modal interactivo con gestión de fallbacks automático.
    *   Visualizador de fragmentos de código interactivo integrado para proyectos 100% backend.
*   **Sección de Estadísticas y Skills:** Gráficos de barra de progreso interactivos que ilustran el dominio en las diferentes áreas tecnológicas.
*   **Diseño Responsive Completo:** Optimizado para pantallas móviles, tablets y ordenadores de escritorio.
*   **SEO Optimizada:** Metadatos estructurados semánticamente para mejorar la indexabilidad en buscadores.
